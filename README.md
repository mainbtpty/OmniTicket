============================================================
OmniTicket v1.0 - Setup & Installation Guide
============================================================

Thank you for your purchase! Follow these simple steps to 
get your premium ticket bot up and running.

Step 1: Install Dependencies
-----------------------------
Make sure you have Python installed on your server/hosting panel.
Open your terminal and run:
pip install -r requirements.txt

Step 2: Add Your Bot Token
---------------------------
1. Open the "main.py" file.
2. Look for your host panel's Environment Variables (Variables section).
3. Add a new key named: DISCORD_TOKEN
4. Paste your Discord Bot Token as the value.

Step 3: Run the Bot
--------------------
Start the application using:
python main.py

Step 4: Configure Inside Discord
---------------------------------
Once the bot is online in your server, use the following slash command 
as an Administrator to deploy the ticket system panels:
/setup_tickets
